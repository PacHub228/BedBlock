# Bedblock

Плагин для сервера Minecraft (на базе Spigot/Paper), который добавляет элемент опасности при попытке сна.

## Описание
Когда игрок пытается лечь в кровать, кровать мгновенно ломается, а игроку отправляется предупреждающее сообщение о том, что спать в этом мире слишком опасно.

## Возможности
*   Автоматическое прерывание попытки сна.
*   Уничтожение кровати (замена блока на воздух).
*   Отправка предупреждающего сообщения игроку.

## Установка
1. Скачайте последнюю версию плагина.
2. Помстите `.jar` файл в папку `plugins` вашего сервера.
3. Перезапустите сервер.
